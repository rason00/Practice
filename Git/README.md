# Git

```Git
git clone xxx          // 克隆库
touch xxx              // 新建xxx文件
git rm xxx             // 删除xxx文件
git status             // 查看状态
git add -A             // 提交所有变化到缓存区
git commit -m '描述'    // 版本描述
git push               // 推送
git pull               // 拉取
git branch             // 查看全部分支
git checkout -b xxx    // 新建xxx分支并跳转
git checkout master    // 跳转到master分支
git branch -d xxx      // 删除xxx分支
git merge xxx          // 合并xxx分支
```
