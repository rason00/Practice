# Git

指令

```
git clone xxx                    // 克隆库
mkdir xxx                        // 新建xxx文件夹
touch xxx                        // 新建xxx文件
git rm xxx                       // 删除xxx文件
git status                       // 查看状态
git add .                        // 提交变化到缓存区
git commit -m '描述'              // 版本描述
git push                         // 推送
git pull                         // 拉取
git branch                       // 查看全部分支
git checkout -b xxx              // 新建xxx分支并跳转
git checkout master              // 跳转到master分支
git branch -d xxx                // 删除xxx分支
git merge xxx                    // 合并xxx分支
git log                          // 查看记录
git remote                       // 查询远端数据库
git remote rename origin xxx     // origin数据库改名为xxx
git push origin xxx              // 推送xxx分支到远端
git stash                        // 暂存内容
git stash pop                    // 还原暂存
git stash clsar                  // 清除全部暂存
```

> 以上是个人整理平时使用的 git 指令（并不完整，且注释或有不恰）
